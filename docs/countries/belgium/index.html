Belgique · version prototype

Cette version V0 utilise le GGMMI/RMMMG comme référence salariale, un taux salarié ONSS de 13,07 % et un taux employeur simplifié de 25 %. Le précompte professionnel et les réductions de cotisations ne sont pas encore simulés.

L’objectif est d’abord de poser une base comparable avec les modules France et Allemagne. Les paramètres belges seront enrichis progressivement.

Méthodologie

Le module Belgique V0 repose sur une simulation simple du coût du travail dans le secteur privé standard. Le salaire brut est calculé comme multiple du GGMMI/RMMMG mensuel.

Le salaire net avant impôt est obtenu en retirant les cotisations sociales salarié. Le coût employeur est obtenu en ajoutant les cotisations employeur au salaire brut.

Limites actuelles : le précompte professionnel, la réduction structurelle, les réductions bas salaires, les différences sectorielles et les effets familiaux ne sont pas encore simulés.
